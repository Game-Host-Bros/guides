This is how you can revert your Euro Truck Simulator 2 game version to play on [Euro Truck Simulator 2 servers](https://www.gamehostbros.com/euro-truck-simulator-2-server-hosting/) that use different versions. Normally when an update releases for the game client, this same update can be delayed for days until SCS also releases updated dedicated server software. Follow these steps to downgrade your Euro Truck Simulator 2 version.

## Downgrading Euro Truck Simulator 2

<Steps>
1. Open Steam.
2. Click Libraries, then right-click `Euro Truck Simulator 2 > Properties`.
3. Select `Betas`.
4. Choose the Beta you'd like to use.
5. Launch Euro Truck Simulator 2.
</Steps>

:::danger
Please be aware that you may experience issues with any game rollback.
:::

:::tip
Do you also want to change your server version? Follow [this guide](/games/euro-truck-simulator-2/change-server-version/) to downgrade your ETS 2 server version.
:::
