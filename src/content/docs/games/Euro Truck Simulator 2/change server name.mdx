This is what you do if you need to change your server name on your Euro Truck Simulator 2 server, also referred to as a hostname or Session Name. This is what is displayed to players in the server browser.

1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Lobby Name` setting and enter your desired server name.
4. Restart your server.

:::note
This setting will override anything you have manually set inside of server.cfg
:::
