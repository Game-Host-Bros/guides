Console in ETS 2 is mainly used for uploading mods to servers as it's required to export your enabled mods, here's how to enable it.

## Enabling Console In ETS 2

<Steps>

1. Press `Windows Key + R` and enter this.
   ```
   %USERPROFILE%\Documents\Euro Truck Simulator 2
   ```
2. Open `config.cfg`
3. Change the existing `uset g_console` and `uset g_developer` lines to `1`. 
   - uset g_console "1"
   - uset g_developer "1"
4. Save the changes.
</Steps>

**macOS File Location**

- ~/Library/Application Support/Euro Truck Simulator 2

**Linux File Location**

- user/.local/share/Euro Truck Simulator 2/

To open the in-game console, press `~` on your keyboard. 

Further in-depth console information can be found [here](https://truck-simulator.fandom.com/wiki/Console).
