Here is how you can connect to your [Euro Truck Simulator 2](https://www.gamehostbros.com/euro-truck-simulator-2-server-hosting/) using a Session Search ID in the in-game server browser.

## Connecting With A Session Search ID :badge[Recommended]{variant=success}

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and restart your Euro Truck Simulator 2 server.
2. In the sidebar, go to `System > Console`.
3. Press `Crtl + F` and search `Session search id` then copy your session search id.
   - Be sure to remove the /101 when searching in-game. 
   - For example `85266395932841714/101` would become `85266395932841714`.
4. Open ETS 2, click `Continue Career > Convoys` from the main menu and then select `Convoy Lobby`.
5. Wait for the list to load, then in the top left paste your session search id and click the search button.
6. Select your server then click the `Join Convoy` button at the bottom of the screen.
</Steps>

## How to Connect Using the Convoy Browser
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and set a name for your Euro Truck Simulator server.
2. Open Euro Truck Simulator 2.
3. Go to the `Convoys` section in the main menu and then select `Convoy Lobby`.
4. Wait for the list to load, then in the top left type your session name and click the search button.
5. Select your server then click the `Join Convoy` button at the bottom of the screen.
</Steps>
