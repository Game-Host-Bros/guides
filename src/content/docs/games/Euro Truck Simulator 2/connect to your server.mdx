Here is how you can join your Euro Truck Simulator 2 dedicated server using the IP address.

## How to Connect Using the Convoy Browser
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and set a name for your Euro Truck Simulator server.
2. Open Euro Truck Simulator 2.
3. Go to the `Convoys` section in the main menu and then select `Convoy Lobby`.
4. Wait for the list to load, then in the top left type your session name and click the search button.
5. Select your server then click the `Join Convoy` button at the bottom of the screen.
</Steps>
