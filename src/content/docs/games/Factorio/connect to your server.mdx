This guide will help you quickly connect to your Factorio dedicated server using an IP address.

## How to Connect Using the In-Game Browser
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Factorio server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Factorio, from the main menu select `Multiplayer > Connect to address`.
4. In the IP address field, paste the copied server IP and click `Connect`.
</Steps>
**Example:** If your server IP is 192.168.1.100:34197, paste:
`192.168.1.100:34197`
