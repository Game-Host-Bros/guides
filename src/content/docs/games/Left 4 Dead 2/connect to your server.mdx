Here is how you can join your L4D2 dedicated server using the IP address.

## How to Connect Using the Console

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your L4D2 server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Left For Dead 2.
4. Open the `Developer Console (~)`, type connect then paste the copied server IP, and press Enter to connect.

:::tip
If the Developer Console doesn't open in game, go to `Options > Keyboard/Mouse` and then enable `Allow Developer Console`.
:::
