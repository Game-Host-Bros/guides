[The Lord of the Rings Return to Moria](https://www.gamehostbros.com/the-lord-of-the-rings-return-to-moria-server-hosting/) includes multiple difficulty presets to quickly confiure your server, this guide will cover using existing presets along with how to use custom difficulty settings.

## Changing Difficulty Presets

:::note
You must create a new world to change difficulty settings. You can do so by changing World Name in Startup Parameters.
:::

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Management > File Manager` and open `MoriaServerConfig.ini`.
3. Under [World.Create] look for `Difficulty.Preset` and enter one of the following presets:
   - **story**, **solo**, **normal** or  **hard**.
4. Go to `Configuration > Startup Parameters`.
4. Change `World Name` to generate a new world.
5. Start your server.
</Steps>

## Custom Difficulty Configuration

Here's how to set custom difficulty settings in LotR Return to Moria, this is allows you to adjust individual settings.

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Management > File Manager` and open `MoriaServerConfig.ini`.
3. Under [World.Create] edit `Difficulty.Preset` to `custom`.
4. Edit your desired custom difficulty settings and click save.
5. Go to `Configuration > Startup Parameters`.
6. Change `World Name` to generate a new world.
7. Start your server.
</Steps>

Here are all of the difficulty settings for LotR Return To Moria, and what each of their possible values are:

| Setting | Possible Values `verylow, low, default, high, veryhigh` |
|---------|-------|
| CombatDifficulty | verylow, low, default, high, veryhigh |
| EnemyAggression | low, default, high, veryhigh |
| SurvivalDifficulty | verylow, low, default, high |
| MiningDrops | verylow, low, default, high |
| WorldDrops | verylow, low, default, high |
| HordeFrequency | verylow, low, default, high, veryhigh |
| SiegeFrequency | verylow, low, default, high, veryhigh |
| PatrolFrequency | verylow, low, default, high, veryhigh |

:::tip
The possible values are `verylow`, `low`, `default`, `high` and `veryhigh`. Not all settings have these parameters, however they will automatically default to the nearest value, for example low or high instead.
:::
