This is what you do if you need to change your server name on your [The Lord of the Rings Return to Moria server](https://www.gamehostbros.com/the-lord-of-the-rings-return-to-moria-server-hosting/), also referred to as a hostname. This is what is displayed to players in the server browser under session history.

## Changing World Name

:::danger
Changing this setting will create a new world.
:::

<Steps>
1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `World Name (Save Name)` setting and enter your desired server name.
4. Restart your server.
</Steps>
