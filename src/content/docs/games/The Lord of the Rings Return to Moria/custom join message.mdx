It's possible to setup a message that displays to players connecting to your [The Lord of the Rings Return to Moria server](https://www.gamehostbros.com/the-lord-of-the-rings-return-to-moria-server-hosting/).

Follow this easy guide to create a custom join message that can be used to display rules and more for your LotR Return to Moria server.

## Editing MoriaServerRules.txt

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Management > File Manager`.
3. Enter your desired message.
4. Click save and restart your server.
</Steps>
