By default, Game Host Bros [The Lord of the Rings Return to Moria servers](https://www.gamehostbros.com/the-lord-of-the-rings-return-to-moria-server-hosting/) will have updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your The Lord of the Rings Return to Moria Server.

## Updating Your The LotR Return to Moria Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Server.
2. Go to `System > Console`.
3. Restart your server.
</Steps>
