This is what you do if you need to set a password on your [The Lord of the Rings Return to Moria server](https://www.gamehostbros.com/the-lord-of-the-rings-return-to-moria-server-hosting/).

## Setting A LotR Return to Moria Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Restart your server.
</Steps>
