This guide will help you quickly connect to your Sons of the Forest dedicated server using an IP address.

## How to Connect Using the In-Game Menu
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Sons of the Forest server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Sons of the Forest.
4. From the main menu, select `Multiplayer > Join Game`.
5. Click `Direct Connect`, paste the copied server IP into the address field, and press `Join`.
</Steps>
**Example:** If your server IP is 192.168.1.100:8765, paste:
`192.168.1.100:8765`
