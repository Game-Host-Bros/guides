Here is how to set a password for your [Sons Of The Forest server](https://www.gamehostbros.com/sons-of-the-forest-server-hosting/).

## Setting Your Sons Of The Forest Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Start your server.
</Steps>
