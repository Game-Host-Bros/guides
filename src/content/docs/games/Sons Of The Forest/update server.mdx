By default, Game Host Bros [Sons Of The Forest servers](https://www.gamehostbros.com/team-fortress-2-server-hosting/) will have updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your SoTF Server.

## Updating Your Sons Of The Forest Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Automatic Update` setting and make sure it's enabled.
4. `Start` your Server.
</Steps>
