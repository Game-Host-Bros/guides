At Game Host Bros, we've integrated basic Astroneer settings directly into our game panel, here's how to quickly and easily configure your new [Astroneer server](https://www.gamehostbros.com/astroneer-server-hosting/).

## Configuring Your Astroneer Server

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your server.
2. Go to `Configuration > Startup Parameters`.
3. Edit the settings you'd like to change.
4. Start your server.
</Steps>
