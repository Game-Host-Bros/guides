Here is how you can connect to your [The Isle Evrima server](https://www.gamehostbros.com/the-isle-evrima-server-hosting/) from the in-game server browser.

:::note
If you haven't already, to connect to your server you must enable the Evrima DLC. 

Do the following:

1. Right-Click `The Isle` in your Steam library.
2. Select `Properties > Betas`.
3. Edit `Beta Participation` and select `evrima - EVRIMA Public Branch`.
4. Start The Isle to finish installing Evrima.
:::
