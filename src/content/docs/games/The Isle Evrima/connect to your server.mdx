Here is how you can connect to your [The Isle Evrima server](https://www.gamehostbros.com/the-isle-evrima-server-hosting/) from the in-game server browser.

:::note
If you haven't already, to connect to your server you must enable the Evrima DLC. 

Do the following:

1. Right-Click `The Isle` in your Steam library.
2. Select `Properties > Betas`.
3. Edit `Beta Participation` and select `evrima - EVRIMA Public Branch`.
4. Start The Isle to finish installing Evrima.
:::

## Setting A Server Name

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Set a unique `Server Name`.
4. Restart Your server.
</Steps>

## Connecting To Your The Isle Evrima Server

<Steps>
1. Open The Isle (Evrima).
2. Click `Play`.
3. Select Session Filter and choose `Unofficial`.
4. Search for your server name.
5. Select your server and click `Connect`.
</Steps>
