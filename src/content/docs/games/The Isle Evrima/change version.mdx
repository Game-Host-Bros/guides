If you don't want to play The Isle Evrima, and would rather play a different beta branch. Here's how to change the version of your [The Isle Evrima server](https://www.gamehostbros.com/the-isle-evrima-server-hosting/). 

## Changing Steam Beta Version

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Edit `Beta Branch`.
4. Go to `Configuration > Server Actions`.
5. Click `Reinstall Server` and wait for the install to finish.
</Steps>

Here are the currently available beta branches to choose from, they can be found here:

- [The Isle Beta Branches](https://steamdb.info/app/412680/depots/).

The main branches are `evrima` and `public`.
