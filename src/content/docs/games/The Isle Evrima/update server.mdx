By default, Game Host Bros [The Isle Evrima servers](https://www.gamehostbros.com/the-isle-evrima-server-hosting/) will have updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your The Isle Evrima Server.

## Updating Your The Isle Evrima Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Automatic Update` setting and make sure it's enabled.
4. Restart your Server.
</Steps>
