At Game Host Bros, we've integrated basic The Isle Evrima settings directly into our game panel, here's how to quickly and easily configure your new [The Isle Evrima server](https://www.gamehostbros.com/the-isle-evrima-server-hosting/).

## Basic Configuration Of Your Server

Basic settings such as Server Name, Max Players and more are integrated directly into our game panel.

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Edit the settings you'd like to change.
4. Restart your server.
</Steps>

## Advanced Configuration

This is how you use the Game.ini file to configure your server.

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Management > File Manager`.
3. Navigate to `/TheIsle/Saved/Config/LinuxServer/Game.ini`.
4. Edit the settings you'd like to change.
5. Start your server.
</Steps>
