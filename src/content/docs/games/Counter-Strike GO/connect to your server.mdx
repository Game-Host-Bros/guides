Here is how you can join your CSGO dedicated server using the IP address.

## How To Install CS:GO

CS2 completely replaced CS:GO in the Steam store, luckily there's still a way to install CS:GO, here's how it's done.

<Steps>
1. Open `Steam > Library`.
2. Click `Counter-Strike 2 > Properties`.
3. Select `Betas > csgo_legacy`.
4. Start the game begin installing.
</Steps>

## How To Connect Using The Console
<Steps>
1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your CSGO server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Counter-Strike Global Offensive.
4. Open the `Developer Console (~)`, type connect then paste the copied server IP, and press Enter to connect.
</Steps>
