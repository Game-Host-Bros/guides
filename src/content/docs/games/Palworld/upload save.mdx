In this tutorial, we will guide you through the process of uploading an existing save file to your Palworld dedicated server.

Step 1: go to: Appdata\Local\PAL\Saved\SaveGames
You may need to allow hidden folders under (View)

Step 2: Stop your server so it isn’t running in your GameHost Bros Panel

Step 3: In the GameHost Bros panel go to 
Go to management on the left has side and click “File Manager”
Once there follow through the files - /home/container/PAL/Save/Savegame/0

Please note: To change these files correctly you must use a FTP program, such as FileZilla. 

The required filezilla server information can be found under STFP details.
