Here is how to kick or ban a player on your Palworld Server.

## How to Kick a Player

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Select your Palworld server.
3. In the Console type `showplayers`
4. Copy the Player's SteamID64 [example:76561198106341655] 
5. type `kickplayer steam_76561198106341655` 


## How to ban a Player

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Select your Palworld server.
3. In the Console type `showplayers`
4. Copy the Player's SteamID64 [example:76561198106341655] 
5. type `banplayer steam_76561198106341655` 

To do this in-game simply become an Admin by typing `/adminpassword [YourAdminPassword]` then `/showplayers` `/banplayer steam_76561198106341655`
