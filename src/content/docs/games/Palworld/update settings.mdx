Whenever Pocketpair releases an update for [Palworld servers](https://www.gamehostbros.com/palworld-server-hosting/) that adds new server settings, in most cases these new settings won't be added to your servers configuration files.

Follow these steps to update your PalWorldSettings.ini file with the newest settings available for Palworld. 

:::note
This is not how to configure your server, this guide covers how to add the newest settings to your server.

To configure your server settings follow our [Palworld Server Configuration Guide](/games/palworld/configure-palworld/).
:::

## Updating Your PalWorldSettings.ini File

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Tools > Mod Manager`.
3. Install `Palworld Config Updater`.
4. Start your server.
</Steps>
