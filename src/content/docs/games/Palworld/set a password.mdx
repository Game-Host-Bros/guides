title: How To Set A Password On your Palworld Server
description: Learn how to set a password on your Palworld server.
tableOfContents: false
sidebar:
  label: Setting a Password
---

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Palworld server.
2. Click `Configuration > Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Go back to the console page, then restart your server.

:::danger
There is currently a bug with some Xbox/Gamepass players where they crash when a password is set on Palworld. We are waiting for the Palworld developers to resolve this.

If you have this issue, you will need to remove your server password.
:::
