Here is how you can join your Palworld dedicated server using the IP address.

## How to Connect to Your Palworld Server

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your Palworld server.
3. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
4. Open Palworld and click `Join Multiplayer Game`
5. At the bottom of the screen you will see `127.0.0.1:8211`, replace this with your IP address and click `Connect`

## Connect To a Palworld Server With a Password

:::caution
Passwords do not current work in Palworld
:::

If your Palworld server has a password set, you will need to add the password to the end of your IP address. It should look like this.

`IP:PORT;password YOURPASSWORDHERE`

another example

`127.0.0.1:8211;password YOURPASSWORDHERE`
