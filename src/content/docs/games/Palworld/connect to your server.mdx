Here is how you can join your [Palworld Server](https://www.gamehostbros.com/palworld-server-hosting/) using the IP address.

## PC Palworld Server 
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Palworld Server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Palworld and click `Join Multiplayer Game`.
4. At the bottom of the screen you will see `127.0.0.1:8211`, replace this with your IP address and click `Connect`.
</Steps>

## Xbox Palworld Server
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Palworld Server.
2. Set a unique [server name](/games/palworld/change-server-name/) in `Configuration > Startup Parameters` and then `Restart` your server.
3. Select `Join Multiplayer Game` in the Palworld menu.
4. Select `Community Servers`.
5. Check the box "Display servers with different versions".
6. Search your Servers name in the box located at the top.
7. Once found, select your server to connect.
</Steps>

If your Palworld server doesn't show up in the browser at all then follow these steps in this order. **Test after each step**.

- Make sure you've changed the server name as per our [guide](https://guides.gamehostbros.com/games/palworld/change-server-name/) and followed all the steps.
- Restart your server and wait 60 seconds.
- Turn on Display servers with different versions.
- Double check you're on the Community Servers tab.
- Change your server name to something simple.
- Restart your game.

If you're still having trouble then please reach out to Game Host Bros support.

:::tip
You can also connect by searching for your [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) rather than the server name.
:::
