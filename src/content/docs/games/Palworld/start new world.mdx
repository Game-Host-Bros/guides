Starting fresh in Palworld can be an exciting opportunity to explore new adventures, build different structures, and experience the game from a whole new perspective. Whether you're dealing with world corruption, or simply looking for a clean slate, resetting your world is quick and straightforward.

<Steps>
1. Login to the Game Host Bros panel and stop your Palworld server.
2. In the Game Host Bros panel go to `Management > File Manager`.
3. Once there navigate to `PAL/Save/Savegame/0`.
4. Delete the folder with random numbers like `84B5E2264EC387DEDB43179D666031A1`.
5. Start your server.
</Steps>
