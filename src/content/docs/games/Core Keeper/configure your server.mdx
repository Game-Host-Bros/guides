At Game Host Bros, we've integrated basic Core Keeper settings directly into our game panel, here's how to quickly and easily configure your new [Core Keeper server](https://www.gamehostbros.com/core-keeper-server-hosting/).

## Configuring Your Core Keeper Server

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Edit the settings you'd like to change.
4. Start your server.
</Steps>
