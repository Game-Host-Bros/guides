This is what you do if you need to change your server name on your [Ark Survival Evolved server](https://www.gamehostbros.com/ark-survival-evolved-server-hosting/), also referred to as a hostname. This is what is displayed to players in the server browser.

## Changing Your Ark Survival Evolved Server Name

<Steps>
1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Ark server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Server Name` setting and enter your desired server name.
4. Restart your server.
</Steps>
