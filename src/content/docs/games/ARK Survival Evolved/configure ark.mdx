Here's how you can locate the and edit the config file for your [Ark Survival Evolved server](https://www.gamehostbros.com/ark-survival-evolved-server-hosting/), this guide covers.

The [GameUserSettings.ini Generator](https://ini.arkforum.de/index.php) is a powerful tool available on ArkForum that simplifies ARK server configuration. Instead of manually adding and configuring every setting this tool includes an easy-to-use interface to generate your settings.

A full breakdown of settings can also be found on the [ARK SE Server Configuration Wiki](https://ark.fandom.com/wiki/Server_configuration#GameUserSettings.ini), this page also contains details on how to configure your Game.ini file, this file is used for advanced configuration of an ARK server.

## Using The GameUserSettings.ini Generator

:::note
Basic settings such as Server Name and Password are set in our game panel under `Configuration > Startup Parameters`.
:::

<Steps>
1. Use the [ARK GameUserSettings.ini Generator](https://ini.arkforum.de/index.php) to set your desired settings.
2. Click `Create File` then copy everything directly under the `[ServerSettings]` section in the generated output.
3. Login to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your Ark server.
4. In File Manager, Navigate to `/ShooterGame/Saved/Config/LinuxServer/GameUserSettings.ini`.
5. Paste the settings you copied earlier under the header of `[ServerSettings]`, replacing what used to be there.
6. Start your server.
</Steps>

## Manually Configuring GameUserSettings.ini

By default, an ARK server won't include all of the settings in your GameUserSettings.ini file. 

A full breakdown of the possible settings can be found on the [ARK SE Wiki](https://ark.fandom.com/wiki/Server_configuration#GameUserSettings.ini).

<Steps>
1. Firstly, login to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your Ark server.
2. On the left sidebar, navigate to `Management` then `File Manager`.
3. Navigate to `/ShooterGame/Saved/Config/LinuxServer/GameUserSettings.ini`.
4. After you have made the changes you want, start your server. 
</Steps>
