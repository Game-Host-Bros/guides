Here is how you can connect to your [Aloft server](https://www.gamehostbros.com/aloft-server-hosting/) using a Friend Code or from the in-game server browser.

## Connecting With A Friend Code :badge[Recommended]{variant=success}

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Aloft server.
2. In the sidebar, go to `Management > File Manager`.
3. Open the file `ServerRoomCode.txt` and copy the numbers.
4. Open Aloft, click `Multiplayer > Join Game` from the main menu.
5. Enter your friend code in the top right and click Join Game.
</Steps>

:::note
Your Friend Code will change every time your server restarts.
:::

## How to Connect Using the Browser
It can sometimes take 15 - 30 mins for your server name to show in the browser. We suggest you use the friend code only.
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and set a name for your Aloft server.
2. Open Aloft, click `Multiplayer > Join Game` from the main menu.
3. Select the region your server is in and look for the name in the list.
</Steps>
