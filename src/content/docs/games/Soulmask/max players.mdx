Need more players? Trying to limit slots? At Game Host Bros we make setting Max Players in Soulmask a quick and easy process, here's how it's done.

1. Login to the [Game Host Bros Panel](https://panel.gamehostbros.com) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Edit `Max Players`.
4. Start your server.
