Here is how you can join your [Soulmask server](https://www.gamehostbros.com/soulmask-server-hosting/) using the IP address.

## Connecting To Your Soulmask Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Soulmask server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Soulmask and click `Online Game > Connect to the server directly`.
4. Enter your IP address, port and click `Confirm`
</Steps>
:::caution
If you've set a password, you must enter it here otherwise the server will kick you back to the main menu.
:::
