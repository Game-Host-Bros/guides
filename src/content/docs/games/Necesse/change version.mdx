If you don't want to play Necesse on the latest public version, and would rather play a different beta branch, here's how to change the version of your [Necesse server](https://www.gamehostbros.com/necesse-server-hosting/). 

## Changing Steam Beta Version

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Edit `Steam Beta Branch`.
4. Start your server.
</Steps>

Here are the currently available beta branches to choose from, they can be found here:

- [Necesse Beta Branches](https://steamdb.info/app/1169370/depots/#branches).

The main branch is `public`.
