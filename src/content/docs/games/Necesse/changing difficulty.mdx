If you're not feeling the current difficulty and would like to change, [Necesse](https://www.gamehostbros.com/necesse-server-hosting/) makes it easy to do so. Follow these simple steps to get it done, whether you want to relax and take it easy on an easier mode, or take on the challenge of Brutal, turning the game into a great, yet unforgiving adventure for you and your friends!

## Changing Difficulty Through Console
<Steps>
  1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
  2. Go to `System > Console`.
  3. Type `Difficulty [Value]`.
  4. Join your server.
</Steps>

The possible Necesse difficulties are as listed here:

| Command      | Difficulty Value |
|--------------|------------------|
| /difficulty  | casual           |
| /difficulty  | adventure        |
| /difficulty  | classic          |
| /difficulty  | hard             |
| /difficulty  | brutal           | 


## Changing Difficulty In-Game

First you must set yourself as an `Admin`, to do this follow our adding admins guide, [here](https://guides.gamehostbros.com/games/necesse/adding-admins/). You have to be Admin or above to execute this command, and you can see a list of all the commands [here](https://guides.gamehostbros.com/games/necesse/adding-admins/#command-list).
<Steps>
  1. [Connect](https://guides.gamehostbros.com/games/necesse/connect-to-server/) to your Necesse Server.
  2. Open chat with `Enter`, and type `/difficulty [Value]`.
</Steps>
The possible Necesse difficulties are as listed here:

| Command      | Difficulty Value |
|--------------|------------------|
| /difficulty  | casual           |
| /difficulty  | adventure        |
| /difficulty  | classic          |
| /difficulty  | hard             |
| /difficulty  | brutal           |
