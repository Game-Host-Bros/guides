Necesse makes it easy to connect to your Server for the first time, and even easier to reconnect, here's how to connect.

## Connecting To Your Server

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Copy the [Server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg).
3. Open Necesse.
4. Click `Multiplayer`.
5. Select `Direct Join`.
6. Enter your Server IP address and Port and click `Connect`.

:::tip
Did you know? Next time you try to connect to your server you can simply click the IP Address in the main menu.
:::
