Here is how you can join your Quake Live dedicated server using the IP address.

## How to Connect Using the Console
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Quake Live server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Quake Live.
4. Press the ~ key on your keyboard to open the `Developer Console`, then type `/connect` followed by your copied server IP, and press Enter to connect.
</Steps>
**Example:** If your server IP is 192.168.1.100:27960, type:
`/connect 192.168.1.100:27960`
