By default, Game Host Bros [American Truck Simulator servers](https://www.gamehostbros.com/american-truck-simulator-server-hosting/) will have updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your American Truck Simulator Server.

## Updating Your American Truck Simulator Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Automatic Update` setting and make sure it's enabled.
4. `Start` your Server.
</Steps>
