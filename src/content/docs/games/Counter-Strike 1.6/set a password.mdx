This is what you do if you need to set a password on your [Counter-Strike 1.6 server](https://www.gamehostbros.com/counter-strike-1-6-server-hosting/).

## Setting Your CS 1.6 Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and open your Counter Strike 1.6 server.
2. Go to `Configuration > Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Go back to the console page, then restart your server.
</Steps>
