To get further help on a specific topic or command, type (without the brackets) `help <topic / command>` into your server console.

## Notation of Command Parameters:
- `<param name>`: Required parameter
- `<entityId / player name>`: Possible types of parameter values
- `[param name]`: Optional parameter
