This is what you do if you need to change your server name on your 7 Days To Die server, also referrered to as a hostname. This is what is displayed to players in the server browser.

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your 7D2D server.
3. Go to `System > Console` and Stop your server.
4. Click on `Management > File Manager.`
5. Open the file `serverconfig.xml` and replace the property value `ServerName` with your own server name.
6. Start your server.

:::note
This setting will override anything you have manually set inside of server.cfg
:::
