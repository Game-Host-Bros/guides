Want to personalize your [7 Days To Die server](https://www.gamehostbros.com/7-days-to-die-server-hosting/) by changing its name? Also known as the hostname, this is what players see in the server browser. This guide will show you how to update your server name quickly and easily through the Game Host Bros Panel.

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your 7D2D server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Server Name` setting and enter your desired server name.
4. Restart your server.
</Steps>
:::note
This setting will override anything you have manually set inside of server.cfg
:::
