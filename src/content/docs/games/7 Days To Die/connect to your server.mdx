Here is how you can join your 7D2D dedicated server using the IP address.

## How to Connect Using the Console

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your 7D2D server.
3. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
4. Open 7D2D.
5. Click `Join A Game` then `Connect To IP`.
6. Enter the server IP address and port from your server and click Connect.
