Here is how you can join your 7D2D dedicated server using the IP address.

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your 7D2D server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open 7D2D.
4. Click `Join A Game` then `Connect To IP`.
5. Enter the server IP address and port from your server and click Connect.
