Here is how you can reset your [7D2D server](https://www.gamehostbros.com/7-days-to-die-server-hosting/) to start fresh and fix performance issues or bugs.

## Resetting Your 7D2D Server

:::caution
This should only be used as a last resort. If you keep having issues, then reach out to a bro. We're here to help you.
:::
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your 7D2D server and stop the server.
3. Go to `Management > File Manager` and delete **ALL the folders and files** (The file manager should have nothing left).
4. In the sidebar select `Configuration > Advanced`.
5. Click `Reinstall Server` and wait for the install to finish.
</Steps>
