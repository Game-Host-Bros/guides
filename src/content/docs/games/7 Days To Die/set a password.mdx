This is what you do if you need to set a password on your 7 Days to Die server.

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your 7D2D server.
3. Go to `System > Console` and Stop your server.
4. Click on `Management > File Manager.`
5. Open the file `serverconfig.xml` and add the property value for `ServerPassword` with your own password. (By default this is empty)
6. Start your server.
