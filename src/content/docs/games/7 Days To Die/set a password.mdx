This is what you do if you need to set a password on your [7 Days to Die server](https://www.gamehostbros.com/7-days-to-die-server-hosting/).

## Setting Your 7 Days To Die Server Password

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your 7D2D server.
2. Click `Configuration > Startup Parameters`.
3. Under `Basic` Look for `Server Password` and then type the password you want.
4. Restart your server.
</Steps>
