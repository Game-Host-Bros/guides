In this guide you will learn how to add admins to your 7D2D server.

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your 7D2D server.
3. In the sidebar click on `Management > File Manager`.
4. Select the `.local/share/7DaysToDie/Saves/` folder.
5. Here, find the file named `serveradmin.xml`.
6. Add a new entry on its own line in SteamID64 format. You can get your SteamID from [here](https://steamid.io/). It should look like this: `<user platform="Steam" userid="76561198106341655" name="[You!]" permission_level="0" />`.
7. Restart your server.
