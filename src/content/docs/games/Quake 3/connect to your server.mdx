This guide will help you quickly connect to your [Quake 3 Arena](https://www.gamehostbros.com/quake-3-arena-server-hosting/) (Q3A) server using an IP address.

## How to Connect Using the Console
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Q3A server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Quake 3 Arena.
4. Open the `Developer Console (~)`, type connect then paste the copied server IP, and press Enter to connect.
</Steps>
**Example:** If your server IP is 192.168.1.100:27960, type:
`connect 192.168.1.100:27960`
