This guide will help you quickly connect to your Rust dedicated server using an IP address.

## How to Connect Using the Console
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Rust server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Rust.
4. Press `F1` to open the console.
5. Type the command `connect` followed by the copied server IP and press `Enter`.
</Steps>
**Example:** If your server IP is 192.168.1.100:28015, type:
`connect 192.168.1.100:28015`
