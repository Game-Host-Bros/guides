Setting a password on your game server is a great way to stop bad actors from joining your server and wreaking havoc. This is how you can set a password on your [V Rising server](https://www.gamehostbros.com/v-rising-server-hosting/).

## Setting Your V Rising Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click `Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Go back to the console page, then restart your server.
</Steps>
