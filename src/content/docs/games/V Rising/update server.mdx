By default, Game Host Bros [V Rising server](https://www.gamehostbros.com/v-rising-server-hosting/) will automatically update whenever a new update releases ensuring you're server is up-to-date in a seamless manner, automatic updates can also be disabled if you want full control over when your server updates. This is how you can update your server after a V Rising update. 

## Updating Your V Rising Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your V Rising server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Automatic Update` setting and make sure it's enabled.
4. Restart your Server.
</Steps>
