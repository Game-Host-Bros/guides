Here is how you can join your [V Rising server](https://www.gamehostbros.com/v-rising-server-hosting/) by using the IP address via Console or the in-game server browser.

## Direct Connecting With The In-Game Server Browser.

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your V Rising server.
2. Copy your [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar. 
3. Open V Rising, and click `Play > Online Play > Show All Servers > Direct Connect`.
4. Paste your server IP address and then click connect.
</Steps>
:::tip
You can also find your server by searching for it's name in the sidebar.
:::

## How To Connect Using The Console
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your V Rising server.
2. Copy your [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar. 
3. In the V Rising main menu, press `~`.
4. Type `connect [ServerIP]`.
</Steps>
