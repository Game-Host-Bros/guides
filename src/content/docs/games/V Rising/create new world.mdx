Here is how you can create a new world on your [V Rising server](https://www.gamehostbros.com/v-rising-server-hosting/) to start completely fresh with a new save, this is helpful in case you've made a mistake that cannot be undone.

:::note
**This is required to access the new 1.1 content for all saves created before the update.**
:::

## Generating A New World

Ensure that your server is stopped before proceeding. Follow these steps to generate a new world.

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Configuration > Startup Parameters`.
3. Change `Save Name`.
4. Start your server.
</Steps>

Alternatively, if you desire a full reset of your server you can follow this guide to reset your server:

- [Reset Server Guide](/games/v-rising/reset-server/)
