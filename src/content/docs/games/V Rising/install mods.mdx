In this guide, you will learn how to install mods and plugins on your [V Rising server](https://www.gamehostbros.com/v-rising-server-hosting/) using [BepInEx](https://thunderstore.io/c/v-rising/p/BepInEx/BepInExPack_V_Rising/).

## Installing BepInEx
Before installing any mods or plugins, you must first install BepInEx
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your V Rising server.
2. In the sidebar, navigate to `Tools > Mod Manager`.
3. Locate BepInEx and click `Install`
</Steps>
:::note
To update BepInEx, simply follow the above process again but click Uninstall and then Install again.
:::

## Installing Plugins
Once BepInEx is installed, you can now add the plugins you wish. We suggest downloading them from [Thunderstore](https://thunderstore.io/c/v-rising/). You will also need to know how to [upload files to your game server.](/getting-started/uploading-files/).
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your V Rising server.
2. Download the plugin you want from [Thunderstore](https://thunderstore.io/c/v-rising/), right click and extract it.
3. Access your V Rising server using [SFTP](/getting-started/uploading-files/).
4. Navigate to `BepInEx/plugins` and upload your extracted plugin folder.
    <FileTree>

   - BepInEx
     - plugins
       - **odjit-KindredPortals** Example plugin.
         - **KindredPortals.dll**
         - **manifest.json**
         - ..   
   </FileTree>
6. Start your server.
</Steps>
If you get stuck at any time, just reach out to a bro on [Discord](https://discord.gg/FN4XgbAwzr) or via a support ticket.
