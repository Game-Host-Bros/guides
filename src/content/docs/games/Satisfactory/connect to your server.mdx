This guide will help you quickly connect to your Satisfactory dedicated server using an IP address.

## How to Connect Using the In-Game Menu

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Satisfactory server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Satisfactory.
4. From the main menu, select `Server Manager > Direct Connect`.
5. In the IP address field, paste the copied server IP and click `Connect`.

**Example:** If your server IP is 192.168.1.100:15777, paste:
`192.168.1.100:15777`
