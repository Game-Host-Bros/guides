This guide will help you quickly connect to your Satisfactory dedicated server using an IP address.

:::caution
When you first connect and setup your Satisfactory server, it may take a few times to connect. This is a bug with Satisfactory.
:::

## How to Connect Using the In-Game Menu

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Satisfactory server.
2. Copy the [server IP address and port](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Satisfactory.
4. From the main menu, select `Server Manager > Add Server`.
5. In the IP address field, paste the copied server IP, replace the port of 7777 with your server port, and click `Connect`.

**Example:** If your server IP is 198.137.23.46:7781, it should look like this:
198.137.23.46
7781
