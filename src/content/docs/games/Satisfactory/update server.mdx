By default, Game Host Bros Satisfactory servers will have automatic updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your Satisfactory Server.
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Automatic Update` setting and make sure it's set to `1`.
4. `Start` your Server.
</Steps>
