This is what you do if you need to set a password on your [Satisfactory server](https://www.gamehostbros.com/satisfactory-server-hosting/).

## Setting Your Satisfactory Server Password

<Steps>
1. Open Satisfactory and click `Server Manager` on the main menu.
2. Select your server.
3. Go to the `Server Settings` tab.
4. Look for `Player Password Protection` and then click `Change Password`.
5. Click `Apply` in the bottom right hand corner.
</Steps>
