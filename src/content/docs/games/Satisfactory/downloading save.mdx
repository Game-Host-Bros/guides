In this tutorial, we will guide you through the process of downloading your save file from your [Satisfactory server](https://www.gamehostbros.com/satisfactory-server-hosting/). This is handy if you need to use a Satisfactory calculator.

## Downloading Saves

<Steps>
1. Firstly, login to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and once there, select your Satisfactory server.
2. Once stopped, on the left, navigate to `Management` then `File Manager`.
3. Navigate to `.config/Epic/FactoryGame/Saved/SaveGames/server` and download the latest auto save file.
</Steps>
