In this tutorial, we will guide you through the process of uploading an existing save file to your [Satisfactory server](https://www.gamehostbros.com/satisfactory-server-hosting/). All of this is now handled in the game itself using the Satisfactory Server Manager.

If you get stuck at any time, Game Host Bros staff can help you. Just reach out to a bro on [Discord](https://discord.gg/FN4XgbAwzr)

## Uploading Existing Save
<Steps>
1. From the Satisfactory main menu, select `Server Manager` and choose the server you wish to upload the save to.
2. Navigate to the `Manage Saves` section.
3. Select the `Upload Save` tab.
4. Click on the `Upload Game` button located at the bottom right.
5. After the upload, go to the `Load Save` tab.
6. Select the save file you uploaded, then click on the `Load Game` button.
</Steps>
