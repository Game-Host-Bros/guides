This is what you do if you need to change your server name on your [No More Room In Hell](https://www.gamehostbros.com/no-more-room-in-hell-server-hosting/) (NMRIH) server, also referred to as a hostname. This is what is displayed to players in the server browser.

## Changing Your No More Room In Hell Server Name
<Steps>
  1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your No More Room in Hell server.
  2. Go to `Configuration > Startup Parameters`.
  3. Find the `Server Name` setting and enter your desired server name.
  4. Restart your server.
</Steps>
