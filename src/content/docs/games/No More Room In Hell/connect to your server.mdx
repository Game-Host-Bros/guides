This guide will help you quickly connect to your No More Room in Hell dedicated server using an IP address.

## How to Connect Using the Console

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your No More Room in Hell server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open No More Room in Hell.
4. Press `~` to open the developer console.
5. Type the command `connect` followed by the copied server IP and press `Enter`.

**Example:** If your server IP is 192.168.1.100:27015, type:
`connect 192.168.1.100:27015`
