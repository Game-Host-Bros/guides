This is what you do if you need to set a password on your [Wreckfest server](https://www.gamehostbros.com/wreckfest-server-hosting/), setting a password is a good way to ensure that only trusted people are able to access your server.

## Setting Your Wreckfest Server Password

<Steps>
1. Log in to the [Game Host Bros Game Panel](https://panel.gamehostbros.com/) and select your Wreckfest server.
2. Go to `Configuration > Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Restart your server.
</Steps>
