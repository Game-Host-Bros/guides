Here is how you can join your Wreckfest dedicated server.

1. Set a unique server name in the [Game Host Bros Panel](https://panel.gamehostbros.com/) using [this guide](https://guides.gamehostbros.com/games/wreckfest/change-server-name/)
2. Open Wreckfest.
3. Click `Multiplayer > Browse Lobbies`.
4. Look for your server name in the list and connect.

:::tip
Press **C** while you hover over your server in the browser to add it to your favorites. This way it will always show up at the top of the list.
:::
