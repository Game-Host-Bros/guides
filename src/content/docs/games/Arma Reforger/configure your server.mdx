At Game Host Bros, we've integrated basic Arma Reforger settings directly into our game panel, here's how to quickly and easily configure your new [Arma Reforger server](https://www.gamehostbros.com/arma-reforger-server-hosting/).

## Basic Settings

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your server.
2. Go to `Configuration > Startup Parameters`.
3. Under `Basic`, edit the settings you'd like to change.
4. Start your server.
</Steps>

## Custom Configuration

:::note
Settings in `Configuration > Startup Parameters` must be set there, and will be overriden if manually set in your configuration file.
:::

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. Go to `Management > File Manager`.
3. Open `config.json`.
4. After changing your desired settings, click save.
5. Start your server.
</Steps>

A full list of settings can be found [here](https://community.bistudio.com/wiki/Arma_Reforger:Server_Config#Summary).
