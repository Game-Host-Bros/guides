This is what you do if you need to set a password on your [Arma Reforger server](https://www.gamehostbros.com/arma-reforger-server-hosting/).

## Setting Your Arma Reforger Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Arma Reforger server.
2. Go to `Configuration > Startup Parameters`.
3. Under `Basic`, look for `Server Password` and then type the password you want.
4. Restart your server.
</Steps>
