This is what you do if you need to change your server name on your [Arma Reforger server](https://www.gamehostbros.com/arma-reforger-server-hosting/), also referred to as a hostname. This is what is displayed to players in the server browser.

## Changing Your Arma Reforger Server Name

<Steps>
1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Arma Reforger server.
2. Go to `Configuration > Startup Parameters`.
3. Under `Basic`, look for the `Server Name` setting and enter your desired server name.
4. Restart your server.
</Steps>
