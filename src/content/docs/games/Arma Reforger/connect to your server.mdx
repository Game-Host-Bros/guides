Here is how you can join your [Arma Reforger server](https://www.gamehostbros.com/arma-reforger-server-hosting/) using the IP address.

## Direct Connecting

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Arma Reforger server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Arma Reforger, and select `Multiplayer`.
4. Press `V` `Direct Join`, and paste your IP address and Port.
5. Click `Confirm`.
</Steps>

**Example:** If your server IP is `198.137.23.46:27030`, it should look like this:
```
198.137.23.46
27030
```

:::tip
You can also search for your [Server Name](/games/arma-reforger/change-server-name/), under the Multiplayer > Community menu.
:::
