By default, Game Host Bros [Arma Reforger servers](https://www.gamehostbros.com/arma-reforger-server-hosting/) will have updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your Arma Reforger Server.

## Updating Your Arma Reforger Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Under `Updates`, find the `Automatic Update` setting and make sure it's enabled.
4. Restart your Server.
</Steps>
