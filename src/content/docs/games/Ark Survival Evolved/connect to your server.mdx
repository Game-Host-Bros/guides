## How to Connect to Your Ark Server from Epic Games using the in-game browser. 

1. Set a unique server name in the [Game Host Bros Panel](https://panel.gamehostbros.com/) using [this guide](https://guides.gamehostbros.com/games/ark-survival-evolved/change-server-name/)
2. Select `Join Ark` in the Ark Survival Evolved menu.
3. Click Session Filter and select `Unofficial`.
4. Search your Servers name in the box located at the top.
5. Once found, select your server to connect.

:::tip
Sometimes it may take a couple of minutes for your server to show up.
:::

Here is how you can join your Ark Survival Evolved Dedicated Server using the IP address.

## How to Connect to Your Ark Survival Evolved Server using the Steam browser.

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click your servers `[ServerIP:Port]` to copy it.
3. Open Steam and at the top click `View > Game Servers`
4. Click `Favorites` and select Ark Survival Evolved as the game.
5. Click the Plus (+) Icon and add your `[ServerIP:Port]`
6. Connect to your server.

:::tip
The in-game server browser can be buggy on Steam. We recommend connecting via IP in the above Steam Browser method instead.
:::

If you're having trouble then please reach out to Game Host Bros support.
