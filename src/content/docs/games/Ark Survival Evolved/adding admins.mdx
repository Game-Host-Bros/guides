Here is how to add admins to your Ark Survival Evolved server.

## Setting An Admin Password

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and open your Ark Survival Evolved server.
2. Click `Configuration > Startup Parameters`.
3. Look for `Admin Password` and then type the password you want.
4. Go back to the console page, then restart your server.

## Make Yourself Admin in Ark Survival Evolved

1. Open Ark Survival Evolved and connect to your server.
2. Press `Tab` twice to open Console.
3. Type `enablecheats [YourAdminPassword]`
4. You may test with `cheat fly`, to disable this type `cheat walk`.


## Whitelisting Admins With AllowedCheaterSteamIDs.txt

1. Connect to your server.
2. Open the larger Console by pressing `Tab` twice.
3. To find your Ark ID type `whoami` in Console.
4. Create the file `AllowedCheaterSteamIDs.txt` In `ShooterGame/Saved/`.
5. List the Ark IDs - One per line, of the players you wish to set as Admin.

:::tip
Did you know? There's also an Admin panel in-game, the command to use it is `showmyadminmanager`.
:::
