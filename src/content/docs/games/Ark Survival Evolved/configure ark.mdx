Here's how you can locate the and edit the config file for your Ark Survival Evolved server.

## Editing GameUserSettings.ini
Editing your GameUserSettings.ini file is the fastest way to modify settings on your Ark Survival Evolved server. Please be aware that basic settings such server name are handled by our game panel itself under `Configuration > Startup Parameters`.
<Steps>
1. Firstly, login to the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Ark server.
2. On the left sidebar, navigate to `Management` then `File Manager`.
3. Navigate to `/ShooterGame/Saved/Config/LinuxServer/GameUserSettings.ini`.
4. After you have made the changes you want, restart the server. 
</Steps>
