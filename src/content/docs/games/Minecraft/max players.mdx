Need more players? Or looking to limit users without creating a whitelist? Here's how you can set Max Players on your Minecraft Server.

1. Log into the [Game Host Bros Panel](https://panel.gamehostbros.com) and `Stop` your server. 
2. Go to `Management > File Manager`.
3. Enter `server.properties`.
4. Edit `max-players=`.
5. Save and Start your server.
