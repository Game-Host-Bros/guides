Connecting to your Minecraft server, whether it be Java Edition or Bedrock, is done by following these steps.

## Finding Your Servers IP Address
<Steps>
  1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
  2. Click on your Server and copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
</Steps>
## Connecting To Your Server
<Steps>
  1. Open Minecraft.
  2. Click `Multiplayer` and select `Add Server`.
  3. Enter the Servers [IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg).
  4. Select your Server and click `Join Server`.
</Steps>
:::tip
If you have a lot of servers in the browser, consider giving setting a name for your server in Minecraft. Servers can also be moved up or down in the list by clicking the arrows on it's icon.
:::
