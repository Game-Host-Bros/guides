If you'd like to change your [Minecraft server's](https://www.gamehostbros.com/minecraft-server-hosting/) world seed, here's where to find the setting and how it's done.

## Changing Level Seed

<Steps>
  1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your Minecraft server.
  2. Go to `Configuration > File Manager`.
  3. Open `server.properties`, edit `level-seed=`.
  4. Edit `level-name` to change world.`
  4. Start your Server.
</Steps>

:::tip
When changing Minecraft Server settings outside of the game, you have to restart your server for the changes to apply.
:::
