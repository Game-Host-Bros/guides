In Minecraft there's a common error where you may get kicked for "Flying is not enabled on this server". Luckily the fix for this is quick and easy, this guide will cover where to find your [Minecraft server](https://www.gamehostbros.com/minecraft-server-hosting/) settings file, and how to enable Flight.

## Enabling Flight

<Steps>
  1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Minecraft server.
  2. Go to `Configuration > Startup Parameters`.
  3. Look for `Allow Flight` and set it to `true`.  
  4. Start your Server.
</Steps>
:::tip
When changing Minecraft Server settings outside of the game or server console, you have to restart your server for the changes to apply.
:::
