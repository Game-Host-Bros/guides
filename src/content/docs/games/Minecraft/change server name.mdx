This is what you do if you want to change the server name of your Minecraft server, also referred to as a hostname. This is what is displayed to players in the server browser.

## How To Change Your Minecraft Server Name
<Steps>
  1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/).
  2. Select your server.
  3. Go to `Management > File Manager`.
  4. Find `server.properties` and open it.
  5. Edit `motd=` to the name you desire.
  6. Restart your server.
</Steps>
