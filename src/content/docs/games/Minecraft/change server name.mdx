This is what you do if you want to change the server name of your [Minecraft server](https://www.gamehostbros.com/minecraft-server-hosting/), also referred to as a hostname. This is what is displayed to players in the server browser.

## Changing Your Minecraft Server Name

<Steps>
  1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your Minecraft server.
  2. Go to `Management > File Manager`.
  3. Open `server.properties`, edit `motd=` and then click save.  
  4. Start your Server.
</Steps>
