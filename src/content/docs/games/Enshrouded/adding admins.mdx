If there's a problem player on your server, becoming an Admin will be an easy way to deal with the issue. It's really simple to do, you'll only have to set an Admin Password and connect to the server with it, here's how.

## How To Set An Admin Password

1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Select and Stop your server.
3. Go to `Configuration > Startup Parameters`.
4. Find `AdminPassword` and enter your desired password.
5. Start your server.

## Becoming An Admin 

1. Open Enshrouded and select your server.
2. Connect with the `AdminPassword` you set earlier.

:::tip
You can open the in-game menu and select permissions to confirm that you're now an Admin.
:::
