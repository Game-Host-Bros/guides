Here is how you can join your Enshrouded dedicated server using the IP address.

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Enshrouded server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Steam and click `View > Game Servers`.
4. In the server browser, click the Plus (+) icon and enter your IP address.
5. Click `Connect`
