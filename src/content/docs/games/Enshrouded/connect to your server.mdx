Here is how you can join your Enshrouded server using the IP address.

## Connecting With The Enshrouded In-Game Server Browser :badge[Recommended]{variant=success}

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Enshrouded server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Enshrouded.
4. Click `Play > Join > Find Server (IPv4 address)` and paste your IP address.
5. Select `Add To Favorites` and then `Join` your server from the list.
</Steps>

## Connecting To Enshrouded With The Steam Browser

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Enshrouded server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Steam and at the top, click `View > Game Servers`.
4. In the server browser, click the Plus (+) icon and enter your IP address.
5. Click `Connect`
</Steps>
