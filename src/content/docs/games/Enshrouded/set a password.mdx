Setting a password on your game server is a great way to stop bad actors from joining your server and wreaking havoc. This is how you can set a password on your [Enshrouded server](https://www.gamehostbros.com/enshrouded-server-hosting/).

## Setting Your Enshrouded Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click `Startup Parameters`.
3. Look for `Server Password` and then type the password you want.
4. Restart your server.
</Steps>

:::note
Your Server Password **Cannot** be the same as your Admin Password.
:::
