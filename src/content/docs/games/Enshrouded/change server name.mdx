This is what you do if you need to change your server name on your Enshrouded server, also referred to as a hostname. This is what is displayed to players in the server browser.
<Steps>
1. Log in to the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Select your Enshrouded server.
3. Go to `Configuration > Startup Parameters`.
4. Find the `Server Name` setting and enter your desired server name.
5. Restart your server.
</Steps>
:::note
This setting will override anything you have manually set inside of enshrouded_server.json
:::
