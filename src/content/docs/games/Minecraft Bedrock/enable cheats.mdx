This guide will cover how to enable cheats on your [Minecraft Bedrock server](https://www.gamehostbros.com/minecraft-server-hosting/) using the Game Host Bros game panel, enabling cheats is required to use special commands on your server.

## Enabling Cheats

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your Minecraft server.
2. Go to `Configuration > Startup Parameters`.
3. Set `Enable Cheats` to True.  
4. Start your Server.
</Steps>

:::note
You will also need to make yourself an OP to use privileged commands on your server.

Here's our guide covering this process: [How To Become OP On Minecraft Bedrock Edition](/games/minecraft-bedrock/adding-admins/)
:::
