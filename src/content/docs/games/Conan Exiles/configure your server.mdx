In this guide, we will show you how you can edit the settings on your Conan Exiles server.

## Configuring Your Conan Exiles Server

<Steps>
1. Firstly, login to the [Game Host Bros Game Panel](https://panel.gamehostbros.com/) and once there, select your Conan Exiles Server.
2. Stop your server before proceeding.
3. Once stopped, on the left, navigate to `Management` then `File Manager`.
4. Navigate to `/ConanSandbox/Saved/Config/WindowsServer/ServerSettings.ini`.
5. After you have made the changes you want, start the server. 
</Steps>
