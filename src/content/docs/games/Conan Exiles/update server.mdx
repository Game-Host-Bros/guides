By default, Game Host Bros [Conan Exiles servers](https://www.gamehostbros.com/conan-exiles-server-hosting/) will have updates enabled and all you'll need to do is restart your server to apply an update, here are the steps to update your Conan Exiles Server.

## Updating Your Conan Exiles Server

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Server.
2. Go to `Configuration > Startup Parameters`.
3. Find the `Automatic Update` setting and make sure it's enabled.
4. `Start` your Server.
</Steps>
