Becoming an Admin on your [Conan Exiles server](https://www.gamehostbros.com/conan-exiles-server-hosting/) is essential for server management, letting you manage players and configure your own customizable server settings with ease, this guide walks you through the step-by-step process to add admins in Conan Exiles.

## Setting An Admin Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and stop your server.
2. In the sidebar, go to `Configuration > Startup Parameters`.
3. Edit the `Admin Password` setting.
4. Start your server.
</Steps>

## Becoming An Admin

Now that you've set your [Admin Password](/games/conan-exiles/adding-admins/) and started your server, these next steps cover how to become a Conan Exiles admin.

<Steps>
1. Connect to your Conan Exiles server.
2. Open the menu with `Esc` and click `Settings > Server Settings`.
3. Click `Make Me Admin` and enter your [Admin Password](/games/conan-exiles/adding-admins/).
</Steps>

Since you're now an admin you can now easily configure your server settings, You now also have access to the Admin Panel. Our guides for server configuration and the Admin Panel can be found here:

- [Conan Exiles Admin Panel Guide](/games/conan-exiles/admin-panel/)
- [Conan Exiles Server Configuration Guide](/games/conan-exiles/configure-your-server/)
