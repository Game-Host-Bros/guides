Here is how you can join your [Conan Exiles dedicated server](https://www.gamehostbros.com/conan-exiles-server-hosting/) using the IP address.

## How to Connect to Your Conan Exiles Server
<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open Conan Exiles and click `Online Game > Connect to the server directly`.
4. Enter your IP address, port and click `Confirm`
</Steps>
