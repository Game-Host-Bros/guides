Here is how you can join your Conan Exiles dedicated server using the IP address.

## How to Connect to Your Conan Exiles Server

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your Conan Exiles server.
3. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
4. Open Conan Exiles and click `Online Game > Connect to the server directly`.
5. Enter your IP address, port and click `Confirm`
