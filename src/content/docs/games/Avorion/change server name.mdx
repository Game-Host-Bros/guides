Want to personalize your [Avorion server](https://www.gamehostbros.com/avorion-server-hosting/) by changing its name? This guide will show you how to update your server name quickly and easily through the Game Host Bros Panel.

## Changing Your Avorion Server Name

<Steps>

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your Avorion server.
2. Go to `Configuration > Startup Parameters`.
3. Under `Basic`, find the `Server Name` setting and enter your desired server name.
4. Start your server.
</Steps>
