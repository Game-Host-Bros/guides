Here is how you can join your [Vintage Story server](https://www.gamehostbros.com/vintage-story-server-hosting/) by using the IP address.

## Direct Connecting Using An IP Address.

:::tip
By default, Vintage Story is whitelist only. You either need to set up or disable the Whitelist before you connect. You can quickly disable it by typing `/serverconfig whitelistmode off` into the server console.
:::

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your Vintage Story server.
2. Copy your [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar. 
3. Open Vintage Story, and click `Multiplayer > Add New Server`.
4. Fill out your server name (This can be anything), ip address and password then click ok.
</Steps>
