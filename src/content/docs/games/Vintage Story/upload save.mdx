In this tutorial, we will guide you through the process of uploading a custom world file to your [Vintage Story server](https://www.gamehostbros.com/vintage-story-server-hosting/). These steps are for both Singleplayer and existing Multiplayer world transfers.

If you get stuck at any time, Game Host Bros offers free save transfers. Just reach out to a bro on [Discord](https://discord.gg/FN4XgbAwzr)

## Finding Your Save Files

<Steps>
1. Open Vintage Story.
2. Click `Singleplayer > Open Saves Folder`.
3. Create a copy of the save you wish to upload.
</Steps>

<FileTree>

- AppData
  - Roaming
    - VintagstoryData
      - VintageStory
        - Saves
          - **serene kingdom story.vcdbs** Example save file.
</FileTree>

## Uploading & Enabling Your World

After following the steps above, here's how to upload and then enable a world on your Vintage Story server.

### Uploading Your World

<Steps>

1. Login to the [Game Host Bros Game Panel](https://panel.gamehostbros.com/) and `Stop` your Vintage Story server.
2. Go to `Management > File Manager`.
3. Navigate to `data/Saves/` and delete any existing files.
4. [Upload](/getting-started/uploading-files/) your `.vcdbs` save file.
</Steps>

Once uploaded your server files should look like this:

<FileTree>

- data
  - Saves
    - **serene kingdom story.vcdbs** Example save file.
</FileTree>

### Enabling Your World

<Steps>

1. Login to the [Game Host Bros Game Panel](https://panel.gamehostbros.com/) and `Stop` your Vintage Story server.
2. Go to `Management > File Manager` and enter the `data` folder.
3. Open `serverconfig.json`.
4. Edit `SaveFileLocation` to match the name of your uploaded save file.
5. Save your changes and then start your server.      
</Steps>

```json title="serverconfig.json"
         "WorldConfig": {
       "Seed": null,
       "SaveFileLocation": "./data/Saves/serene kingdom story.vcdbs",
       ```
