This is what you do if you need to set a password on your [Vintage Story server](https://www.gamehostbros.com/vintage-story-server-hosting/).

## Setting Your Vintage Story Server Password

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and open your Vintage Story server.
2. In the server console, type `/serverconfig password YourPasswordHere`.
</Steps>
