In this guide, we will show you how to change the gameplay mode on your [ASKA server](https://www.gamehostbros.com/aska-server-hosting/). 

This is required to use custom settings instead of normal settings.

## Changing Your ASKA Server Mode

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and `Stop` your server.
2. In the sidebar, click `Configuration > Startup Parameters.
3. Edit `mode` to either `normal` or `custom`.
4. Start your server.
</Steps>
