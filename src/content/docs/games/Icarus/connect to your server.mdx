Here is how you can join your [ICARUS server](https://www.gamehostbros.com/icarus-server-hosting/) using the IP address.

## Direct Connecting

<Steps>
1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/) and select your ICARUS server.
2. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
3. Open ICARUS, click `Play` and choose a character.
4. Then from the menu, select `Join`, in the upper-right click `Dedicated Servers (Beta) > Direct Connect`.
5. In the IP address field, paste the copied server IP and click `Confirm`.
6. Select your server in the list to connect.
</Steps>

At first you will be prompted to create a Prospect (Save File), continue following the menu to setup a world and join your server.
