## How to Connect Using the Console

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your CS2 server.
3. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
4. Open Counter-Strike 2.
5. Navigate to `Settings`, then to the `Game tab'.
6. If not already done, activate the `Developer Console (~)' by setting it to 'YES'.
7. Open the console, type connect then paste the copied server IP, and press Enter to connect.
